To-Do App

How It Works

Welcome to the To-Do App! This application is designed to help you manage your tasks efficiently. With a simple and intuitive interface, you can easily add new tasks, mark them as completed, and delete them when they're no longer needed.

Here's a quick guide on how to use the app:

  • Add a Task: Type your task in the input field and click the button to add it to the list.
  • Mark as Completed: Once you finish a task, click the button to mark it as completed. Completed tasks will have a strikethrough effect, making it clear which ones are done.
  • Delete a Task: If a task is no longer needed, you can remove it by clicking the button.
  • Filter Tasks: Use the dropdown menu to filter your tasks based on their status (All, Completed, or Uncompleted).
